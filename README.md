# 🎬 CineAI Studio

Create cinematic AI videos from simple text scripts.

## 🚀 What is CineAI Studio?

CineAI Studio is an AI-powered video creation platform that helps creators transform text scripts into engaging videos with AI voiceovers, subtitles, music, and cinematic effects.

## ✨ Features

* 🎥 Script to Video Generator
* 🎤 AI Voiceovers
* 📝 Auto Subtitles
* 🎵 Background Music
* 🎬 Cinematic Effects
* 💰 Earn Credits
* 🤝 Refer & Earn

## 🎯 How It Works

1. Sign Up
2. Login
3. Paste Your Script
4. Choose Style & Voice
5. Generate Video
6. Download & Share
